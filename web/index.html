AryaCore

A minimalist, high-performance Node.js framework built for modern web applications with zero dependencies

Lightning Fast Ultra Lightweight Zero Dependencies Node.js 18+

Installation

Get started with AryaCore by installing it via npm or yarn:

npm install aryacore
# or
yarn add aryacore

Quick Start

Create your first AryaCore application in just a few lines of code:

const { createApp } = require('aryacore');
const app = createApp();

// Simple route
app.get('/', (req, res) => {
  res.send('Hello World!');
});

// Route with parameters
app.get('/users/:id', (req, res) => {
  res.json({ 
    userId: req.params.id,
    query: req.query 
  });
});

// Start server
app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});

Features

AryaCore comes packed with everything you need to build modern web applications without the bloat.

Blazing Fast

Built directly on Node.js native http module for maximum performance and minimal overhead.

Lightweight

Zero dependencies means minimal footprint, faster installs, and no dependency vulnerabilities.

Full Routing

Supports all HTTP methods with route parameters, query strings, and flexible route patterns.

Middleware

Flexible middleware system with async support for request processing pipelines.

Error Handling

Built-in centralized error handling with customizable error responses.

TypeScript Ready

Includes TypeScript definitions out of the box for better developer experience.

Examples

Explore common patterns and usage examples to get the most out of AryaCore.

Basic Route

Define simple routes with different HTTP methods:

app.get('/about', (req, res) => {
  res.send('About Page');
});

app.post('/login', (req, res) => {
  // Handle login logic
  res.json({ success: true });
});

app.put('/users/:id', (req, res) => {
  // Update user
  res.send(`User ${req.params.id} updated`);
});

app.delete('/posts/:id', (req, res) => {
  // Delete post
  res.status(204).end();
});

Middleware

Create and use middleware functions for cross-cutting concerns:

// Logger middleware
app.use((req, res, next) => {
  console.log(`${req.method} ${req.url} - ${new Date().toISOString()}`);
  next();
});

// Auth middleware
const requireAuth = (req, res, next) => {
  const authHeader = req.headers.authorization;
  
  if (!authHeader || authHeader !== 'Bearer valid-token') {
    return res.status(401).json({ error: 'Unauthorized' });
  }
  
  next();
};

// Apply middleware to specific routes
app.get('/dashboard', requireAuth, (req, res) => {
  res.send('Admin Dashboard');
});

// Global middleware (applies to all routes)
app.use(requireAuth);

Error Handling

Centralized error handling for consistent error responses:

// Throwing errors in routes
app.get('/products/:id', (req, res) => {
  const product = db.getProduct(req.params.id);
  
  if (!product) {
    throw new Error('Product not found');
  }
  
  res.json(product);
});

// Custom error handler
app.onError((err, req, res) => {
  console.error(err.stack);
  
  const statusCode = err.statusCode || 500;
  const message = statusCode === 500 ? 'Internal Server Error' : err.message;
  
  res.status(statusCode).json({
    error: message,
    timestamp: new Date().toISOString(),
    path: req.url
  });
});

// Async error example
app.get('/async-error', async (req, res) => {
  throw new Error('Async error example');
});
